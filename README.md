# Unlimited Tetris

A modern, boundless Tetris game built with vanilla HTML, CSS, and JavaScript.

## Features
- **Unlimited Levels**: Speed increases successfully as you clear lines.
- **Rich Aesthetics**: Neon glow effects, glassmorphism UI, and smooth animations.
- **AdSense Integration**: Dedicated responsive placeholder for Google AdSense implementation.
- **Mobile Responsive**: Adapts to different screen sizes.

## How to Play
- **Arrow Keys / WASD**: Move and Rotate
- **Space**: Pause / Resume
- **C** or **Shift**: Hold Piece

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. (Optional) To enable ads, edit `index.html` and uncomment the AdSense section, replacing `YOUR_AD_CLIENT_ID` with your actual ID.

## Technologies
- HTML5 Canvas
- CSS3 (Variables, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
